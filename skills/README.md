# Skills

OpenHarmony Skills 按公共能力和领域能力两个命名空间组织。

| 命名空间 | 目录 | 定位 |
| --- | --- | --- |
| common | [common/](common/) | 跨领域、跨子系统、跨团队复用的通用能力 |
| domain | [domain/](domain/) | 面向具体框架、子系统或技术域的专用能力 |

新增或迁移 Skill 时必须遵守仓库根目录的命名空间与目录放置规范：

[../openharmony-skills-namespace-and-placement-spec.md](../openharmony-skills-namespace-and-placement-spec.md)
