# 控件视觉识别指南

本文档记录常见无文字标识控件的视觉特征，帮助在调试阶段通过截图快速识别和定位控件。

## 定位优先级原则

> **核心原则：能通过控件树里的文字直接找到控件，就尽量不要用坐标！**

1. `BY.text()` - 文字定位（最优先）
2. `BY.key()` - 唯一标识符
3. `BY.type()` - 控件类型
4. 坐标定位 - 备选方案

## 互动类控件

### 点赞按钮
- 图标: ❤️ 心形形状
- 位置: 通常在内容区域下方
- 定位: `BY.text("点赞")`

### 收藏按钮
- 图标: ⭐ 五角星形状
- 位置: 与点赞按钮相邻
- 定位: `BY.text("收藏")`

### 分享按钮
- 图标: ↗️ 弯曲箭头
- 位置: 内容区域右下角
- 定位: `BY.text("分享")`

### 关注按钮
- 图标: ➕ 加号 + 人形
- 位置: 用户头像旁边
- 定位: `BY.text("关注")`

## 输入类控件

### 搜索框
- 特征: 圆角矩形框，放大镜图标 🔍
- 位置: 页面**顶部**，搜索按钮**左边**
- 类型: `TextInput`, `Search`, `SearchField`
- 定位优先级:
  1. `BY.key('__SearchField__*')`
  2. `BY.type('SearchField')`

### 搜索按钮
- 特征: 按钮形状，有**"搜索"文字**
- 位置: 搜索框**右边**
- 定位: `BY.text('搜索')` ⭐推荐

### 评论输入框
- 特征: 方形气泡，占位文字"说点什么"
- 位置: 页面**底部**
- 类型: `RichEditor`, `TextInput`

## 导航类控件

### 底部导航栏
- 位置: 屏幕**最底部**
- 特征: 3-5 个图标 + 文字标签

### 顶部导航栏
- 位置: 屏幕**最顶部**
- 常见元素: 返回按钮、标题、搜索、更多菜单
