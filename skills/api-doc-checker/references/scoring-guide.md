# 置信度与优先级

## 置信度等级

| 置信度 | 含义 | 建议操作 |
|--------|------|----------|
| 95%+ | 确定 | 立即修复 |
| 85-94% | 高 | 大概率需要修复 |
| 70-84% | 中等 | 建议人工确认 |
| <70% | 低 | 仅供参考 |

## 优先级等级

| 优先级 | 含义 | 处理建议 |
|--------|------|----------|
| Critical | 严重 | 合并前必须修复 |
| High | 高 | 强烈建议修复 |
| Medium | 中 | 建议修复 |
| Low | 低 | 可选改进 |
