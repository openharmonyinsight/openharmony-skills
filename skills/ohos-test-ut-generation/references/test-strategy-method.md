# 测试策略与设计方法

## 代码特征 → 测试策略映射表

| 代码特征      | 生成策略           | 测试场景覆盖                 |
| --------- | -------------- | ---------------------- |
| 多条件判断逻辑   | 条件组合 / MC/DC   | 每个条件独立影响结果，覆盖条件组合      |
| 状态机/多状态逻辑 | 状态迁移路径         | 覆盖所有状态转换：初始→状态A→状态B→终态 |
| 多输入组合规则   | 判定表 / 正交组合     | 系统化覆盖输入组合场景            |
| 资源生命周期    | 创建/失败/释放/重复调用  | 创建成功、创建失败、释放、重复释放      |
| 回调或异步操作   | 成功、失败、未触发、重复触发 | 回调成功、回调失败、未触发、多次触发     |
| 异常处理逻辑    | 正常路径 + 异常路径    | 正常执行、异常抛出、异常捕获         |
| 边界值处理     | 边界值分析          | 最小值、最大值、边界±1、空值        |

## 测试场景划分速查

| 场景类型 | 说明        | 测试等级建议 |
| ---- | --------- | ------ |
| 正常路径 | 正常输入，正常输出 | Level1 |
| 异常路径 | 异常输入，错误处理 | Level2 |
| 边界条件 | 边界值输入     | Level2 |
| 状态相关 | 特定状态下的行为  | Level2 |
| 并发场景 | 多线程/并发情况  | Level2 |

## 参数边界值设计

| 参数类型     | 正常输入       | 边界测试点                   |
| -------- | ---------- | ----------------------- |
| int/long | 典型正值、负值、零值 | INT_MIN, INT_MAX, 0, -1 |
| string   | 非空字符串      | 空字符串、超长字符串              |
| 指针       | 有效对象指针     | nullptr                 |
| 容器       | 非空容器       | 空容器、单元素                 |

## 测试设计输出模板

```markdown

| 编号 | 场景类型 | 场景描述 | 输入数据 | 预期输出 | 测试等级 |
| T01 | 正常 | [具体场景] | [参数值] | [返回值/状态] | Level1 |
| T02 | 异常 | [错误场景] | [触发错误的输入] | [错误码/false] | Level2 |
| T03 | 边界 | [边界值] | [INT_MAX/空字符串等] | [预期行为] | Level2 |
| T04 | 状态 | [状态依赖] | [前置状态描述] | [状态变化结果] | Level2 |
```
