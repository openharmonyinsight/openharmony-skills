# Screen Window Size Test Prompts

用于验证 `screen-window-size` 的三层测试能力。

## 1. Smoke Prompt

一个平板详情页在旋转和拖拽窗口后布局没有刷新，需要补窗口尺寸监听和同步机制。请给出开发阶段的处理建议。

## 2. Scene-Functional Prompt

请为平板详情页补齐窗口尺寸监听和断点同步方案，要求旋转和拖拽窗口后布局立即刷新，并给出可落到工程中的开发建议。

## 3. Real-World Prompt

电商平板商品详情页在竖屏时是单栏，横屏和平行窗口时应切成双栏。但线上回归发现：旋转后偶现还停留在旧布局，拖拽自由窗口缩窄后右侧推荐区也不收起。请输出修复方案，并给出工程验证要求。

## 典型验证点

- 是否命中 `SIZE-01`
- 是否避免误路由到自然方向语义场景
- 是否明确编译、测试和运行时验证要求
