# Avoid Areas Test Prompts

用于验证 `device-avoid-areas` 的三层测试能力。

## 1. Smoke Prompt

线上反馈：评论输入框在弹出软键盘后被完全遮住，同时顶部挖孔区域也压住了标题。请给出问题修复思路。

## 2. Scene-Functional Prompt

请为评论输入区补充软键盘避让能力，要求键盘弹出后输入框始终可见，顶部标题也不能进入挖孔区。请输出可落到工程中的开发方案。

## 3. Real-World Prompt

直播页线上问题：用户发送评论时，软键盘弹出会把输入框完全顶出屏幕，部分机型顶部标题又被挖孔区压住。修完后还要保证沉浸式背景继续铺满。请输出修复方案和工程验证要求。

## 典型验证点

- 是否命中 `AVOID-04`
- 是否将软键盘遮挡归为主问题
- 是否联动窗口尺寸场景作为次级场景
- 是否同时定义工程编译和运行时验证要求
