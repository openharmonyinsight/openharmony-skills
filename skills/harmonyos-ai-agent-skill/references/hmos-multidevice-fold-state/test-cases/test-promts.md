# Fold State Test Prompts

用于验证 `device-fold-state` 的三层测试能力。

## 1. Smoke Prompt

折叠屏应用在悬停态下，上下两个区域的内容被折痕切开，需要重新设计折叠态布局和折痕避让方式。请先给出需求分析设计。

## 2. Scene-Functional Prompt

请为折叠屏视频详情页设计悬停态布局：上半屏播放视频，下半屏承载评论和操作区，并保证退出悬停态后恢复常规布局。请输出可落到工程中的开发方案。

## 3. Real-World Prompt

视频详情页在大折叠设备上展开态正常，但用户折到悬停态后出现三个问题：播放器被折痕切到一半、评论区滚动位置丢失、恢复展开态后播放器重新加载。请给出修复方案，并定义工程验证要求。

## 典型验证点

- 场景命中：`FOLD-02 / FOLD-03 / FOLD-06`
- 阶段输出：`DEV / FIX`
- 证据完整性：`FoldStatus`、方向日志、截图、布局树
- 是否给出折叠态下的编译和回归验证要求
