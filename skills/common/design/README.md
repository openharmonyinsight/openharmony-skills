# 系统设计 (System Design)

架构设计、接口设计、UML 建模、设计文档生成相关技能。

## Skills

| Skill | 说明 |
| --- | --- |
| `ohos-design-agent-instruction-quality-review` | 评审 AGENTS.md / CLAUDE.md / agent instruction 文件的代码地图、知识路由、约束边界和验证闭环质量 |
| `ohos-design-arkts-api-review` | 检视鸿蒙 ArkTS API 接口设计的规范性，覆盖命名、接口设计、同步异步、异常处理、类型数据、注释、事件命名、结构定义等 54 条规则 |
| `ohos-design-c-api-review` | 检视鸿蒙 C API 接口设计的规范性，覆盖命名、接口设计、指针/缓冲区、异常处理、类型数据、注释、结构定义等 50 条规则 |
