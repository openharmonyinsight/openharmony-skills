# Eval 016: annotations 与 TypeScript decorators

## User Prompt

ArkTS 的 annotation 和 TypeScript decorator 是一样的吗？我能不能按 TypeScript decorator 的运行时语义来解释 ArkTS annotation？

## Evaluation Criteria

1. 必须明确指出不能混用两者语义
2. 必须说明 ArkTS annotations 有独立的规范和编译期处理规则
3. 必须提醒不能按 TypeScript decorator 推断运行时行为
4. 必须引用 `spec/annotations.md`
