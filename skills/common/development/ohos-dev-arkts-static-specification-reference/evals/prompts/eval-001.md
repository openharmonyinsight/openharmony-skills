# Eval 001: ArkTS Any 类型与 TypeScript any 的区别

## User Prompt

ArkTS 中的 `Any` 类型和 TypeScript 中的 `any` 类型是一样的吗？
我能不能把 `Any` 当成 TypeScript 的 `any` 来用？

## Evaluation Criteria

1. 必须明确指出 ArkTS `Any` 是预定义 nullish 类型
2. 必须指出 `Any` 没有方法和字段
3. 必须指出 `Any` 是所有类型的超类型
4. 必须引用 spec/types.md
5. 不得混淆 ArkTS `Any` 与 TypeScript `any`
