# 公共技能 (Common Skills)

跨子系统、跨领域的通用技能，按软件开发生命周期组织。

## 分类

| 阶段 | 目录 | 说明 |
|------|------|------|
| 需求分析 | [requirements/](requirements/) | PRD 分析、需求抽取与评审 |
| 系统设计 | [design/](design/) | 架构设计、接口设计、设计文档生成 |
| 编码开发 | [development/](development/) | 代码审查、代码生成、编码规范、编译构建 |
| 自测试 | [testing/](testing/) | 单元测试、XTS 测试、测试用例生成 |
| CI/CD | [cicd/](cicd/) | CI 诊断、构建流水线、PR 工作流 |
| 问题分析 | [troubleshooting/](troubleshooting/) | 错误定位、稳定性分析、内存泄漏检测 |
