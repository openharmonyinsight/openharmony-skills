## Claude API 开发（claude-api 技能）
当用户需要使用 Claude API 或 Anthropic SDK 构建应用时使用此技能。

### 触发场景
- 代码中导入 `anthropic`、`@anthropic-ai/sdk` 或 `claude_agent_sdk`
- 用户要求使用 Claude API、Anthropic SDK 或 Agent SDK
- 涉及 Claude 模型集成开发

### 不触发场景
- 代码使用 OpenAI 或其他 AI SDK
- 通用编程任务
- 机器学习/数据科学任务

### 能力
- Claude API 调用方式和最佳实践
- Anthropic Python/TypeScript SDK 用法
- Agent SDK 集成指导
- 流式响应、工具调用等高级功能
