# 集合 (Collections)

## 中文描述
仓颉标准库提供丰富的集合类型:Array(固定大小数组)、ArrayList(动态数组)、HashMap(哈希映射)、HashSet(哈希集合)等。所有集合类型都是泛型的,支持迭代器模式。

## English Description
Cangjie standard library provides rich collection types: Array (fixed-size array), ArrayList (dynamic array), HashMap (hash map), HashSet (hash set), etc. All collection types are generic and support iterator pattern.

## 使用场景
- 数据存储和管理
- 集合操作(增删改查)
- 迭代遍历
- 泛型容器

## 相关实现
- 标准库实现
- 编译器支持在 Sema 模块
- 关键类: ArrayType, HashMapType, HashSetType

## 概念关系图谱

- **同义词**: 集合, Array, ArrayList, HashMap, HashSet, 数组, 列表, 映射, 集
- **相关概念**: generic, iterator, for-loop, 容器, container
- **相关模块**: 无
