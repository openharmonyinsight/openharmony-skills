## 电子表格处理（xlsx 技能）
当用户需要处理电子表格文件（.xlsx、.xlsm、.csv、.tsv）时使用此技能。

### 触发场景
- 打开、读取、编辑或修复电子表格文件
- 创建新的电子表格或从其他数据源生成
- 添加列、计算公式、格式化、图表、数据清洗
- 转换表格文件格式
- 清理或重构混乱的表格数据

### 不适用场景
- 主要产出物是 Word 文档、HTML 报告或 Python 脚本
- Google Sheets API 集成

### 能力
- 创建和编辑 .xlsx、.xlsm 文件
- 读取和解析 .csv、.tsv 文件
- 公式计算、数据格式化、图表生成
- 数据清洗和结构重组
- 格式转换
